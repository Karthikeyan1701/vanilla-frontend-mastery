🚀 Vanilla Frontend Mastery
    -> This repository is designed as a comprehensive refresher program to revisit and strengthen core frontend development fundamentals using pure web technologies.

    -> The focus is on rebuilding strong practical knowledge in:
        (1) HTML
        (2) CSS
        (3) JavaScript
        (4) TypeScript

    -> The goal is to develop deep understanding of browser behavior, UI logic, performance, and scalable frontend architecture before relying on frameworks.

📌 Purpose
    -> Modern frameworks are powerful, but true frontend expertise comes from understanding the core technologies underneath them.
    -> This repository focuses on:
        (1) Writing clean, semantic HTML
        (2) Creating responsive and accessible layouts
        (3) Mastering CSS layouts and animations
        (4) Understanding JavaScript deeply
        (5) Applying TypeScript for scalable frontend applications
        (6) Building UI logic from scratch

🧠 Learning Roadmap
    ✅ HTML (Semantic elements, Form validation, Accessibility (ARIA basics), SEO fundamentals, Media Handling)
    🎨 CSS (Box model & positioning, Flexbox & Grid, Responsive design, Animations & transitions, CSS architecture (BEM concepts), Modern CSS features)
    ⚡ JavaScript (ES6+ fundamentals, DOM manipulation, Event handling, Async programming (Promises, Async/Await), Browser APIs, Modular JavaScript, Performance optimization)
    🔷 TypeScript (Static typing fundamentals, Interfaces & types, Generics, DOM typing, Scalable project structure, Type-safe UI logic)
